É possível usar cupons promocionais disponibilizados por lojas ou pela plataforma.
Um cupom gerado por uma loja é disponível apenas em compras dentro daquela lojas.
Um cupom gerado pela plataforma é disponível em todas as compras de todas as lojas.
Um cupom só pode ser ser utilizado uma vez por usuário.

Para um cupom válido é preciso armazenar:
#### Obrigatórios
- Nome para identificação;
- Descrição para identificação e apresentação;
- Código promocional para reconhecimento e uso do cupom;
- Taxa promocional oferecida pelo cupom para aplicação em pedidos;
- Status atual para validação de atividade;
- Origem do cupom, que pode ser a loja especifica que o gerou ou a própria plataforma;
