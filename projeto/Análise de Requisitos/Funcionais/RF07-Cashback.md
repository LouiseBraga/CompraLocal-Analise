Todo cliente tem direito a cashback depois de fazer um pedido.
A taxa do cashback depende do valor total do pedido.

Uma ocorrência de cashback válida deve armazenar:
#### Obrigatórios
- Pedido para identificação;
- Valor devolvido para operação;
